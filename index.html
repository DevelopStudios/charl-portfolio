Virtual AHU Dashboard

React· TypeScript· FastAPI· WebSocket

Operator console for a virtual BACnet air-handling unit — first project outside Angular. React hooks drive a shared polling + WebSocket data layer against a FastAPI backend: live point overrides, priority arrays, and alarm acks against a real device model instead of mock data.

Ledger Stream

Angular· RxJS· WebSockets· WebLLM

Real-time crypto feed streamed into a 10,000-row virtual grid. An adjustable RxJS throttle decouples render pressure from the socket; an in-browser LLM flags significant moves — no server, no backend, no API key.

Source Live preview
